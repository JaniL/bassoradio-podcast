## How to run?

Install bower, node and npm

```shell
# First install bower and npm

# Then install packages:
$ npm install && bower install

# Run dev-server
$ grunt serve

# Profit!

# If you want to deploy use:
$ grunt build

# Now in ./dist folder contains built project ready to deploy into interwebs

```
